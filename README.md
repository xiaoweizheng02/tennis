# Tennis Coach 智能网球教练网站

一个专业的网球技术分析和指导网站，帮助网球爱好者提升技术水平。

## 功能特性

- 🎥 **视频分析**：上传网球视频，获得专业的动作分析和改进建议
- 📝 **文字描述分析**：通过文字描述获得技术指导
- ❓ **网球知识问答**：解答网球相关问题
- 📱 **响应式设计**：支持手机、平板和桌面设备

## 技术栈

- 前端：HTML5, CSS3, JavaScript
- 后端：静态网站（可扩展）
- 部署：GitHub + Render

## 如何使用

1. 访问网站
2. 选择分析方式（视频或文字）
3. 上传视频或输入文字描述
4. 获得专业分析报告
5. 根据建议进行练习

## 本地运行

```bash
# 克隆项目
git clone <repository-url>

# 打开 index.html 文件
open index.html
```

## 部署

1. 推送到 GitHub 仓库
2. 在 Render 上创建新的静态网站
3. 连接 GitHub 仓库
4. 部署网站
